{% if request.session.language == 'en' %}1. Arithmetic Brownian Motion (ABM){% else %}1. Movimento Browniano Aritmético (MBA){% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} The Arithmetic Brownian Motion is defined by the stochastic differential equation: {% else %} O Movimento Browniano Aritmético é definido pela equação diferencial estocástica: {% endif %}

$$ dX_t = \mu \, dt + \sigma \, dW_t $$

{% if request.session.language == 'en' %} Here, the drift $\mu$ and volatility $\sigma$ are absolute constants, independent of the current value $X_t$. Increments are additive and identically distributed over equal time intervals. The value at any time $t$ follows a normal distribution: {% else %} Aqui, a deriva $\mu$ e a volatilidade $\sigma$ são constantes absolutas, independentes do valor atual $X_t$. Os incrementos são aditivos e identicamente distribuídos em intervalos de tempo iguais. O valor em qualquer tempo $t$ segue uma distribuição normal: {% endif %}

$$ X_t \sim \mathcal{N}\!\left(X_0 + \mu\, t,\;\; \sigma^2 t\right) $$

{% if request.session.language == 'en' %} Unlike GBM, ABM allows negative values, making it suitable for interest rates, temperature, or profit/loss in absolute terms. The key contrast with GBM is that in GBM both parameters are proportional to $S_t$: {% else %} Ao contrário do MBG, o MBA permite valores negativos, tornando-o adequado para taxas de juros, temperatura ou lucro/perda em termos absolutos. O contraste principal com o MBG é que no MBG ambos os parâmetros são proporcionais a $S_t$: {% endif %}

$$ dS_t = \mu\, S_t\, dt + \sigma\, S_t\, dW_t $$

{% if request.session.language == 'en' %}2. Generalized Brownian Process (Wiener){% else %}2. Processo Browniano Generalizado (Wiener){% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} The Wiener process, also known as standard Brownian motion $W_t$, is a continuous-time stochastic process that models random movement. It is characterised by independent, normally distributed increments with mean zero and variance equal to the time interval. A generalised Brownian process adds a constant drift and volatility: {% else %} O processo de Wiener, também conhecido como movimento Browniano padrão $W_t$, é um processo estocástico de tempo contínuo que modela movimento aleatório. É caracterizado por incrementos independentes e normalmente distribuídos, com média zero e variância igual ao intervalo de tempo. Um processo Browniano generalizado adiciona deriva e volatilidade constantes: {% endif %}

$$ dX_t = \mu \, dt + \sigma \, dW_t $$

{% if request.session.language == 'en' %} where $\mu$ is the drift and $\sigma$ is the volatility. In finance, it forms the basis for modelling the random component of asset prices, where price changes are assumed to follow a random walk. {% else %} onde $\mu$ é a deriva e $\sigma$ é a volatilidade. Em finanças, ele forma a base para modelar o componente aleatório dos preços dos ativos, onde as mudanças de preço são assumidas como seguindo um passeio aleatório. {% endif %}

{% if request.session.language == 'en' %}3. Generalized Brownian Process (Itô){% else %}3. Processo Browniano Generalizado (Itô){% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} Itô's Lemma is a fundamental result in stochastic calculus that extends the chain rule to stochastic processes. For a function $f(X_t, t)$ where $dX_t = \mu\, dt + \sigma\, dW_t$, it states: {% else %} O Lema de Itô é um resultado fundamental no cálculo estocástico que estende a regra da cadeia para processos estocásticos. Para uma função $f(X_t, t)$ onde $dX_t = \mu\, dt + \sigma\, dW_t$, ele afirma: {% endif %}

$$ df = \left(\frac{\partial f}{\partial t} + \mu \frac{\partial f}{\partial X} + \frac{1}{2}\sigma^2 \frac{\partial^2 f}{\partial X^2}\right) dt + \sigma \frac{\partial f}{\partial X}\, dW_t $$

{% if request.session.language == 'en' %} This is crucial for deriving models like Black-Scholes and enables the analysis of asset prices that follow Geometric Brownian Motion. {% else %} Isso é crucial para derivar modelos como Black-Scholes e permite a análise de preços de ativos que seguem o Movimento Browniano Geométrico. {% endif %}
