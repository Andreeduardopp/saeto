{% if request.session.language == 'en' %} Mathematical Structure — ABM: The Arithmetic Brownian Motion is defined by the SDE dXt = μ dt + σ dWt, where drift μ and volatility σ are absolute constants (not proportional to the current value). This leads to a normal distribution for Xt at any time t, with E[Xt] = X0 + μt and Var[Xt] = σ²t. In contrast, GBM uses dSt = μ St dt + σ St dWt, where both parameters are proportional to the current price, yielding a log-normal distribution. {% else %} Estrutura Matemática — MBA: O Movimento Browniano Aritmético é definido pela EDE dXt = μ dt + σ dWt, onde deriva μ e volatilidade σ são constantes absolutas (não proporcionais ao valor atual). Isso leva a uma distribuição normal para Xt em qualquer tempo t, com E[Xt] = X0 + μt e Var[Xt] = σ²t. Em contraste, o MBG usa dSt = μ St dt + σ St dWt, onde ambos os parâmetros são proporcionais ao preço atual, gerando uma distribuição log-normal. {% endif %}
