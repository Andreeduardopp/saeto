Crank-Nicolson Method

{% if language == 'en' %} The Crank-Nicolson method is a finite difference method used for solving the heat equation and similar partial differential equations. It is second-order accurate in both time and space. {% else %} O método Crank-Nicolson é um método de diferenças finitas usado para resolver a equação do calor e EDPs similares. Ele possui precisão de segunda ordem tanto no tempo quanto no espaço. {% endif %}
